Portfólio profissional desenvolvido com React e Vite, com foco em performance, responsividade e otimização para SEO. A interface moderna foi construída com TailwindCSS, e um componente 3D interativo foi implementado com Three.js, proporcionando uma experiência visual envolvente. O projeto é versionado com Git, hospedado na Vercel e utiliza domínio personalizado, garantindo uma presença online sólida, rápida e profissional.

O portfólio foi desenvolvido com React.js, aproveitando sua modularidade e eficiência na construção de interfaces dinâmicas. A estilização foi feita com TailwindCSS, proporcionando um desenvolvimento rápido, responsivo e altamente customizável com classes utilitárias no próprio markup. Enquanto o elemento 3D, foi criado com Three.js, enriquecem a experiência visual e destaca o projeto com um toque moderno e tecnológico. O resultado é uma aplicação responsiva e otimizada para performance e SEO.
